# Route Filters
Use route filters to modify the default handling of routes. A route filter recieves the route fragment and can perform actions before or after the route is 
executed. Route filters can be added via the [Location.routeFilters collection](location-routeFilters.html).

## Methods

**add**  
Use this method to add a route filter.

**remove**  
Use this method to remove a route filter.
