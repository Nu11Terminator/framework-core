# get

Use the *get* method to retrieve a model from a collection.


## Syntax

`get(id)`


## Parameters

**id**  
The ID of the model the *get* command is retrieving.


## Return value
None


## Errors thrown
None


## Example

```javascript
var userInfo = collection.get("321YWCG_0000L0VMP00000E");
```
